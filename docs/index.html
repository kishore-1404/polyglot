Built for live voice

This demo mirrors the realtime pipeline: audio in, streaming response out, and TTS playback with echo suppression and latency overlays.

Streaming TTS

Sentence-level buffering sends partial responses to speech while the LLM continues generating.

Barge-in aware

High-confidence speech interrupts playback for natural turn taking.

Latency telemetry

TTFT, LLM total, TTS synth, and end-to-end timing indicators stay visible.

Realtime flow

Each stage is tuned for low delay and clarity between listening and speaking.

1. Listen
VAD frames capture speech and build the utterance buffer.
2. Respond
LLM streams JSON. Only the response field goes to TTS.
3. Speak
Audio plays while mic is gated, then echo frames are drained.

Get started

Drop this page into the docs folder and enable GitHub Pages from the main branch. Update the commands below to match your repo.

git clone https://github.com/your-org/polyglot-live cd polyglot-live ./start_servers.sh open http://localhost:7860
Hosting tip: In GitHub settings, set Pages to serve from the docs folder on the main branch.
